Shopify POS Extension

The Shelva POS Extension brings label control directly into Shopify POS, allowing store teams to manage labels instantly while working on the sales floor.

Real-time label status inside POS

The Shelva POS extension attaches to each product's page in POS and displays everything staff need to know about each label, turning POS into a live "label dashboard" for each product.

When viewing a product in Shopify POS, the extension displays:

Connected label barcode
Pairing status
Published status
Battery level
A quick-flash button

This gives staff immediate clarity about the state of each label.

Empowers Staff to Take Action Instantly

A modal opens with all the main label management tasks. Everything works seamlessly without leaving the POS interface.

Pair a new label - Choose size, enter a barcode, or scan one
Flash a label - Pick color and duration
Swap labels - Move labels between products without going to Shopify Admin
Republish - Push the latest image from Shelva to the tag

Runs Fully Inside Shopify's Native Workflows

No logging in. No switching apps. Staff tap a product in POS and manage its label like any other product setting.

Immediate Feedback

Staff see label status in real time without leaving POS. Perfect for quick checks while working on the floor.

Speed in Store

Flash, swap, or pair labels while walking the floor. No need to go back to a computer or open another app.
